# 🚗 Expressway Pro Toolkit v1.0

A modern desktop application built with **CustomTkinter** that helps drivers calculate trip statistics such as **average speed, travel time, and estimated fuel cost**, along with trip logging and export features.

## ✨ Features

- 🚗 Vehicle type selection (Car, Van, Multi-Axle)
- ⏱️ Trip time calculation (Entry & Exit time)
- 📏 Distance-based average speed calculation
- ⚠️ Speed limit warning system
- ⛽ Fuel cost estimation
- 📊 Live trip history logging
- 💾 Export trip logs to `.txt` file
- 🎨 Modern dark UI using CustomTkinter

---

## 🛠️ Built With

- Python 3.x
- CustomTkinter
- Tkinter (messagebox)
- Datetime module

## 📦 Installation

### 1. Clone the repository
```bash
git clone https://github.com/your-username/expressway-pro.git
```
### 2. Install dependencies
```bash
pip install customtkinter
```

▶️ Run the Application
```
python main.py
```

## 🖥️ How It Works
1. Enter trip distance (km)
2. Enter entry & exit time (HH:MM format)
3. Set vehicle type and fuel settings
4. Click Calculate Trip Data
5. View:
    - Average speed
    - Safety status (Safe / Speeding)
    - Estimated fuel cost
6. Export trip history if needed
